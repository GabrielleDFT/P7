# Groupomania

## Installation 
### Pré-requis
- Node.js 
- MongoDB
- Express
- Bcrypt
- Dotenv

## Lancement de l'application : Backend
- 1. Tapez dans le Terminal "npm install" pour installer les dépendences & le dossier node_modules
- 2. cd backend / npm start
