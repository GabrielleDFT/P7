# Groupomania

## Installation 
### Pré-requis
- Node.js 
- MongoDB
- Express
- Bcrypt
- Dotenv

## Lancement de l'application : Backend
- 1. Dans le Terminal, tapez "npm install" pour installer le dossier node_modules : cd backend / npm i
- 2. Mettre les données des variables d'environnement
- 3. cd backend / npm start
